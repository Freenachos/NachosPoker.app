# NachosPoker.App

Free poker tools and calculators.

## Pages

- `/` - Home
- `/variance` - Variance Calculator
- `/winrate` - Win Rate Dashboard
- `/seat` - Seat Selection Tool
- `/profits` - Profits Calculator
- `/articles` - Strategy Articles

## Development

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub and connect to Vercel for automatic deployments.
